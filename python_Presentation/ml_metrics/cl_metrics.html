但是AUC到底代表了什么?

AUC的概率解释

AUC是随机抽取一个正样本和负样本，正样本的评分>负样本的评分的概率

假设:

  • $A$是模型预测为真正样本的点的评分的分布.
  • $B$是模型预测为真负样本的点的评分的分布.
  • $\tau$为阈值

$TPR=P(A>\tau),\ FPR=P(B>\tau)$

$AUC = \int_0^1 TPR(x)dx = \int_0^1 P(A>\tau(x))dx$, 这里$x$是FPR

如果认为$x \sim U[0,1)$, $AUC = E_x[P(A>\tau(x))] \tag{1}$

$x=FPR = P(B>\tau(x))$

$P(B>\tau(x)) \sim U$

$=> P(B<\tau(x)) \sim (1-U) \sim U$

$\begin{equation}=> F_B(\tau(x)) \sim U \tag{2}\end{equation}$

$F_X(X)=P(F_X(x)<{X})$

$=P(X\lt F_X^{-1}(X))$

$=F_XF_X^{-1}(X)=X$

得到$\tau(x) \sim B$

最后：$AUC=E_x(P(A>B))=P(A>B)$
