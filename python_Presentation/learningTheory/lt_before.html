Hypothesis Class

  • The Hypothese Class $\mathcal{H}$ is a space of functions.
  • The learning algorithm learns a function (hypothesis) $h\in\mathcal{H}$
  • 我们假设$D$为具有$N$个数据点的数据集，训练样本记为$(x_n,y_n)_{n=1}^{N}$，符合分布$P(x,y)$，也记为$D\sim P^N$

Train Error and Expect Error

0-1 Train Error


$\mathcal{L}_D(h)=\frac{1}{N}\sum_{n=1}^{N}\mathcal{I}(h(x_n)\neq y_n)$

0-1 Expect Error/True Error


$\mathcal{L}_P(h)=E_{(x,y)\sim P}[\mathcal{I}(h(x_n)\neq y_n)]$

在一般情况下，Expect Error > Train Error

我们想要知道Expect Error和Train Error之间会相差多少？

Hoeffding不等式


如果$X_1,X_2,...,X_n$为一组独立同分布的参数为 $p$ 的伯努利分布随机变量，$n$ 为随机变量的个数。定义这组随机变量的均值为：

$$\bar{X}=\frac{X_1+X_2+...+X_n}{n}$$

对于任意 $\delta\gt0$ ，Hoeffding不等式为：

$$P(|\bar{X}-E(\bar{X})|\ge\delta)\le 2exp(-2\delta^2n)$$

训练误差与测试误差的距离上界

对于整个假设空间，也就是这 $M$ 个假设函数，可以得到：


$$P[(|L_{P}(h_1)-L_{D}(h_1)|>\epsilon) \cup (|L_{P}(h_2)-L_{D}(h_2)|>\epsilon)...(|L_{P}(h_m)-L_{d}(h_m)|>\epsilon)]$$

$$\le P[|L_{P}(h_1)-L_{D}(h_1)|>\epsilon] + P[|L_{P}(h_2)-L_{D}(h_2)|>\epsilon]+ ...+ P[|L_{P}(h_m)-L_{D}(h_m)|>\epsilon]$$

$$\le 2M exp(-2\epsilon^2 N)$$

对于$h\in \mathcal{H}:$

$P[(L_{P}(h)-L_{D}(h))>\epsilon] \le exp(-2\epsilon^2 N)$

令 $M exp(-2\epsilon^2 N)\le \delta$ (failure probability)

$\delta$ 即训练误差和测试误差的差异 $\epsilon \ge \sqrt{(log(M)+log(\frac{1}{\delta}))/2N}$ 的概率

得到：$N \ge\frac{1}{2\epsilon^2}(log(M)+log(\frac{1}{\delta}))$

在 $1-\delta$ 的概率下，$L_P(h)\le L_D(h)+\sqrt{\frac{log(M)+log(\frac{1}{\delta})}{2N}}$

上式表明：当 $N$ 足够大时，train loss 和 true loss将非常接近

  • Question: 如果 $M$ 是无限大，怎么办？
